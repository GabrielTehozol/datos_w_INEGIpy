Obtención de datos con INEGIpy¶

  • Elaborado por: Gabriel Tehozol
  • Contacto: gabrieltehozol@gmail.com

Trabajar en Colab implica el uso de recursos computacionales alojados en la nube, especficamente en los servidores de Google Cloud Platform (GCP). Esta plataforma, proporcionada de manera gratuita por Google, permite a los usuarios crear y ejecutar cdigo en un entorno de Jupyter Notebook alojado en lnea en la nube de GCP. Dicha implementacin permite que los recursos computacionales, como CPU, GPU y RAM, sean proporcionados por los servidores de Google, de tal manera que el usuario solo requiere de una conexin a Internet y un navegador web para acceder a la plataforma y trabajar en sus proyectos. Adems, el trabajo en la nube permite a los usuarios acceder a sus proyectos desde cualquier dispositivo y lugar, compartir fcilmente sus proyectos con otros y aprovechar la escalabilidad y flexibilidad de los recursos disponibles en la nube.

Bajo esta misma idea, podemos trabajar con INEGIpy y de esta manera llevar los recursos necesarios directamente a colab.

Instalamos e importamos las librerías pertinentes¶

Inicialmente solo se tendra que instalar geopandas e INEGIpy, sin embargo, en ciertas ocaiones se requieren otras paqueteras que facilitaran el funcionamiento de las libreras, al estar trabajando en la nube, esto debe realizarse cada ocasin que se se trabaje con el cuaderno de trabajo.

Que es INEGIpy?

INEGIpy es una libreria, la cual facilita la descarga y procesamiento de datos del Instituto Nacional de Estadstica y Geografa (INEGI) de Mxico; permite el acceso a datos tanto estadsticos como geogrficos de Mxico. Puedes encontrar mayor informacin en INEGIpy.

La librera funciona a travs de una API (Application Programming Interface) lo cua se traduce como Interfaz de Programacin de aplicaciones. Esta se utiliza para generar conexiones en los programas utilizados. Para acceder a la API propia de INEGI es necesario obtener el token, este es utilizado para autorizar solicitudes de conexin entre el distribuidor de datos y el solicitandte de los datos. Puedes obtener tu token aqu.

Descarga y tratamiento de datos¶

Ahora podemos obtener datos de INEGI, en este caso ejemplificaremos con el Producto Interno Bruto ("PIB"). La variable "Indicadores" que hemos definido previamente como "ind" tiene diversas funciones entre ellas se encuentra la de "obtener_df()", dentro de las cuales se deben cubrir al menos 4 variables primordiales (indicadores, nombres, inicio y fin). En indicadores debemos especificar la clave de la variable que estamos buscando, en "nombres", colocar el nombre que deseemos asignar a las variables asignadas, y tanto en inicio como en fin colocar las fechas de comienzo y termino de la serie que deseamos obtener.

In [9]:
# Definimos las variables primordiales
indicadores = ["6207061841"]        # El indicador del PIB es "6207061841"
nombres = ["PIB"]                   # El nombre nuestra variable ser PIB 
inicio = '2010-01'                  # Definimos la fecha de comienzo de nuestra serie que para este ejemplo sera el 2010
fin = '2020- 12'                    # Definimos la fecha de termino de nuestra serie que para este ejemplo ser 2020
# Solicitamos las variables previas 
pib = ind.obtener_df(indicadores = indicadores,
